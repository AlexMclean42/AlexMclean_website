SQL & Python Database Application

This project is a database management system for an art museum. It allows users to browse and search the collection, manage art objects and artists, and manage users. Admins and users can also manipulate data in the database by inserting, updating, or deleting records. The project is implemented in Python using the mysql-connector-python library to connect to a MySQL database. The script provides a menu-based interface that allows users to select different options and perform various tasks, such as writing queries, browsing the collection, and manipulating data in the database.

How to run the code

  1. Open the Art-museum.py file into your text editor.
  2. Open the Art-museum.sql file into your SQL workbench(I used MySQLworkbench).
  3. User must install Tabulate in their terminal using the command:
    pip install tabulate
  4. User must install mysql.connector in their terminal using the command:
    pip install mysql-connector-python
  5. Users must login to the SQL database using their personal mySQL username and password. This is only required when accessing the admin and user access levels.
  6. When viewing tables in the the VS code terminal make sure that the screen is full screen (double click the terminal and select toggle size to content width).
  7. Click the play button or use the shortcut "command + R" to run the code.
  8. Follow the prompts in the console.

How the code works

The code first establishes a connection to a MySQL database using the mysql-connector-python library. The user is prompted to enter their login information, either as an admin or a guest. If the user is an admin, they must enter a username and password to connect to the database. If the user is a guest, they are connected to the database using predefined login credentials. After the connection to the database is established, the script displays a menu of options depending on the user's login status. The user can choose to write a query, manipulate data in the database, manage users, browse the collection, or quit. The script provides several functions that allow the user to search for or browse records in different tables in the database, and it also provides options for admins and users to manipulate data in the database by entering the data manually or by providing a SQL script file. The script uses the mysql-connector-python library's cursor object and the execute method to execute SQL statements and commit the changes to the database.
